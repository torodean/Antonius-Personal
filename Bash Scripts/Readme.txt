README for discribing Bash script usage and purpose.

# Note: To run these scripts, the EOL conversion may need to be set for UNIX.


CloneAll.sh - Clones all git repositories. Setup for personal usage.
gitRefresh.sh - Refreshes all cloned git repositories. Setup for personal usage and may need edited based on environment.
Reset_Wifi.sh - Resets the wifi adaptor on an Ubuntu machine.
